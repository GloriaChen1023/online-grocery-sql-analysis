# Online Grocery SQL Analysis

## Overview
This project demonstrates SQL-based exploratory data analysis on transactional retail data and translates raw data into actionable business insights.

## Objectives
- Analyze customer purchasing behavior  
- Identify top-selling products  
- Evaluate brand performance  
- Understand sales trends over time  

## Methods
SQL techniques used:
- JOIN (multi-table analysis)
- GROUP BY (aggregation)
- Subqueries (top orders analysis)
- Filtering (time-based analysis)
- Ranking logic for top products and brands

## Key Analyses
- Total number of customers and orders  
- Average order value  
- Top 25 products by units and revenue  
- Customer-level purchase behavior  
- Top-performing brands overall and in the last 6 months  
- High-value customer identification  

## Key Insights
- Sales are highly concentrated in a small number of products  
- A small group of customers contributes disproportionately to revenue  
- Brand performance varies significantly over time
